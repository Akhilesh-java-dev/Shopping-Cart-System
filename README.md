# Shopping Cart System

A Java console-based Shopping Cart application built using Object-Oriented Programming (OOP).

## Features
- View Products
- Add Products to Cart
- Quantity Selection
- Bill Generation
- Automatic Discount Calculation

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Loops
- Switch Case

## Project Overview
This project allows users to view products, purchase items, calculate total bills, and apply discounts automatically through a menu-driven interface.
